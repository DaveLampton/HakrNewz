# HakrNewz

Demo GraphQL Backend using:

- Node.js
- graphql-yoga
- Prisma ORM
- PostgreSQL

Features:

- GraphQL endpoint
- Authentication
- Realtime PubSub
- Multiple models with CRUD queries
  - Filtering
  - Pagination
  - Sorting

Run:

- npm start

Browse to http://localhost:4000 to use GraphQL Playground.
